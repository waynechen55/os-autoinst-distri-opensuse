test_containered_app

 test_containered_app($runtime, [$buildah], $dockerfile, $base);

Create a container using a dockerfile and run smoke test against that. base can be used to setup base repo in the dockerfile in case that the file does not have a defined one.

The main container runtimes do not need buildah variable in general, unless you want to build the image with buildah but run it with $<runtime>
